# 🏫 نظام إدارة الحالات النفسية المدرسية - Frontend

> الواجهة الأمامية (واجهة المستخدم) الخاصة بنظام إدارة الحالات النفسية في المدارس. هذا الجزء هو تطبيق React.js يستخدم Vite كأداة بناء.

---

## 📋 المتطلبات الأساسية

قبل البدء، تأكد من تثبيت ما يلي على جهازك:

| البرنامج | الإصدار | رابط التحميل |
|----------|---------|-------------|
| Node.js | v18 أو أحدث | [nodejs.org](https://nodejs.org) |
| npm | v9 أو أحدث | (يأتي مع Node.js) |

---

## 🚀 خطوات التشغيل المحلي

### 1. استنساخ المشروع

```bash
# افتح سطر الأوامر (Terminal) وانتقل للمجلد الرئيسي
cd project_nafsiaty/frontend
```

### 2. تثبيت الحزم Dependencies

```bash
npm install
```

### 3. إعداد ملف البيئة (.env)

أنشئ ملف جديد باسم `.env` في مجلد `frontend` (إذا لم يكن موجوداً) وضع فيه:

```env
# رابط الـ Backend API
VITE_API_URL=http://localhost:5000/api
```

> **ملاحظة:** إذا كان الـ Backend يعمل على منفذ مختلف أو على خادم بعيد، عدّل هذا الرابط.

### 4. تشغيل التطبيق

```bash
# وضع التطوير (مع إعادة التحميل السريع عند التعديل)
npm run dev

# أو لفتحه في المتصفح مباشرة
npm run dev -- --open
```

سترى رسالة类似于:
```
  VITE v5.x.x  ready in 300 ms

  ➜  Local:   http://localhost:5173/
```

## 🌐 للوصول من أجهزة أخرى في نفس الشبكة

إذا أردت اختبار التطبيق على هاتفك أو جهاز آخر متصل بنفس WiFi:

```bash
# اكتشف عنوان IP الخاص بجهازك
# Windows: ipconfig
# Mac/Linux: ifconfig

# ثم شغل التطبيق بالأمر:
npm run dev -- --host 0.0.0.0
```

ثم افتح على الجهاز الآخر: `http://YOUR_IP_ADDRESS:5173`

## 🔧 استكشاف الأخطاء

### خطأ "Cannot connect to server"

1. تأكد من أن الـ Backend يعمل على المنفذ 5000
2. تحقق من `VITE_API_URL` في ملف `.env`
3. تأكد من عدم وجود جدار حماية يحجب الاتصال

### خطأ "Module not found"

```bash
# أعد تثبيت الحزم
rm -rf node_modules package-lock.json
npm install
```

### الشاشة بيضاء (White Screen)

1. افتح أدوات المطور (F12) في المتصفح
2. تحقق من تبويب Console للأخطاء
3. غالباً يكون السبب خطأ في API URL

## 📱 الأدوار والصلاحيات

### 👨‍💼 المدير (Admin)
- إدارة المستخدمين (إضافة/تعديل/حذف)
- إدارة الطلاب والفصول
- متابعة المدفوعات
- مراقبة الدردشة

### 👩‍🏫 المستشار النفسي (Counselor)
- فتح ومتابعة الحالات النفسية
- جدولة الجلسات الإرشادية
- كتابة التقارير والنصائح
- التواصل مع المعلمين وأولياء الأمور

### 👨‍🏫 المعلم (Teacher)
- تسجيل الملاحظات السلوكية
- إرسال التقارير
- التواصل مع المستشار وأولياء الأمور
- طلب تدخل教务

### 👨‍👩‍👧 ولي الأمر (Parent)
- متابعة حالة ابنه النفسية
- استلام التقارير
- طلب الجلسات الإرشادية
- الدفع عبر الإنترنت
- التواصل مع المدرسة
